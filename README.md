# COCO Parking

Desktop application for parking management: vehicle entries/exits, till, tariffs, debtors, metrics, roles, and backup.

- **Stack:** Tauri 2, React, TypeScript, Vite, SQLite
- **Languages:** English, Spanish (i18n)

## Download installer

You can install COCO Parking without cloning the repository:

- **[Latest release](https://github.com/dministrador/coco_parking/releases/latest)** — Download the Windows installer (.exe or .msi) from the latest release.
- **[All releases](https://github.com/dministrador/coco_parking/releases)** — See every version and choose the installer you need.

## Quick start

```bash
cd app
npm install
npm run tauri:dev
```
